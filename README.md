# Stats Fútbol Clasificación

Aplicación interactiva desarrollada en **Streamlit** para el **análisis estadístico de equipos de fútbol**  y **predecir el rendimiento deportivo de sus jugadores dado sus atributos fisicos y deportivos**.  
Permite explorar métricas clave, visualizar correlaciones entre variables y datos historicos de las temporadas.

**App desplegada:** [https://stats-futbol-clasificacion.onrender.com](https://stats-futbol-clasificacion.onrender.com)
